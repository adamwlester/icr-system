Plotting Tetrode Tracks

Before using this script the following must be performed

1) Export Paxinos Rat Atlas image to image format.
2) Snip image from bregma x,y zero coordinates -12 A-P and -12 D-V
3) Reshape image so that width = hight
4) Convert image to black and white (grayscale) and save it as a bmp file.
5) Name image using the format "Sag_[0.00].png" where values inside brackets indicate position.
6) Set peramiters in for 'par.atlsImageDir' and 'par.atlsImageFile' and run
   in TT_TRACK.m
